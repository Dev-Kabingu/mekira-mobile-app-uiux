# Mekira Online Shopping UI/UX

This repository showcases the design work for the Mekira online shopping site, emphasizing usability and branded aesthetics.

## Features & Contributions
- Homepage & Product Listings – Designed clean layouts with intuitive navigation and clear product categorization.  
- Shopping Cart & Checkout Flow – Streamlined user journey for easy purchasing with minimal steps.  
- Brand Integration – Applied consistent colors, typography, and iconography to strengthen brand identity.  

## Highlights
- Focused on mobile-first, responsive design for accessibility.  
- Created interactive prototypes in Figma to validate user flows.  
- Iterative refinement based on feedback and benchmarking against leading e-commerce platforms.  

## Purpose
To demonstrate e-commerce UI/UX design expertise, showcasing how minimal layouts and branded visuals enhance the shopping experience.
